## About the Project
**Aurora** is a suite of foundational libraries and ecosystems built for the **Austral** programming language. 

## Motivation
The motivation behind this project is to provide the Austral community with production-ready, highly secure, and well-architected packages that fully leverage Austral's unique capability-based security model and strict linear type system. We aim to establish the definitive standard for systems-level, leak-proof, and memory-safe software development.

## The Vault
The core component of this repository is the **[Aurora Vault](./vault/)**. 

The Vault contains highly secure packages for critical backend operations such as database transactions, token authentication, financial balance constraints, and secure memory handles. 
For a complete list of available Vault packages and the strict architectural development standards required for contribution, please see the **[Vault Documentation](./vault/README.md)**.
