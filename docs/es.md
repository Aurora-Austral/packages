## Acerca del Proyecto 💡
**Aurora** es una suite de bibliotecas y ecosistemas fundamentales construidos para el lenguaje de programación **Austral**.

## Motivación 🔥
La motivación detrás de este proyecto es proporcionar a la comunidad de Austral paquetes listos para producción, altamente seguros y bien arquitectados que aprovechen al máximo el modelo único de seguridad basado en capacidades y el estricto sistema de tipos lineales de Austral. Nuestro objetivo es establecer el estándar definitivo para el desarrollo de software a nivel de sistemas, a prueba de fugas y con seguridad de memoria.

## El Vault (Bóveda) 📦
El **Vault** contiene **[paquetes](../vault/)** altamente seguros para operaciones críticas de backend, como transacciones de bases de datos, autenticación de tokens, restricciones de saldo financiero y manejadores de memoria seguros.
Para obtener una lista completa de los paquetes disponibles en el Vault y los estrictos estándares de desarrollo arquitectónico requeridos para contribuir, consulte la **[Documentación de Paquetes](../vault/README.md)**.

## Código de Conducta 🤝
Estamos comprometidos a proporcionar una comunidad acogedora, inclusiva e inspiradora para todos los desarrolladores.
- **Respeto**: Trate a todos con respeto, profesionalismo y empatía.
- **Comentarios Constructivos**: Las discusiones técnicas deben ser rigurosas, de mente abierta y siempre constructivas.
- **Excelencia**: Nos esforzamos por alcanzar los más altos estándares absolutos de calidad de código, seguridad y experiencia del desarrollador.
