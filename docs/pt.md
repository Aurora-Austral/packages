## Sobre o Projeto 💡
**Aurora** é uma suíte de bibliotecas e ecossistemas fundamentais construídos para a linguagem de programação **Austral**.

## Motivação 🔥
A motivação por trás deste projeto é fornecer à comunidade Austral pacotes prontos para produção, altamente seguros e bem arquitetados, que aproveitem totalmente o modelo de segurança único baseado em capacidades e o rigoroso sistema de tipos lineares do Austral. Nosso objetivo é estabelecer o padrão definitivo para o desenvolvimento de software em nível de sistema, à prova de vazamentos e com segurança de memória.

## O Vault (Cofre) 📦
O **Vault** contém **[pacotes](../vault/)** altamente seguros para operações críticas de backend, como transações de banco de dados, autenticação de tokens, restrições de saldo financeiro e manipuladores de memória seguros.
Para obter uma lista completa dos pacotes disponíveis no Vault e os rigorosos padrões de desenvolvimento arquitetural exigidos para contribuição, consulte a **[Documentação dos Pacotes](../vault/README.md)**.

## Código de Conduta 🤝
Estamos comprometidos em fornecer uma comunidade acolhedora, inclusiva e inspiradora para todos os desenvolvedores.
- **Respeito**: Trate a todos com respeito, profissionalismo e empatia.
- **Feedback Construtivo**: As discussões técnicas devem ser rigorosas, com a mente aberta e sempre construtivas.
- **Excelência**: Nós buscamos os mais altos padrões absolutos de qualidade de código, segurança e experiência do desenvolvedor.
